# storyblok-design-system

Storyblok's Design System

## Project setup
```sh
yarn install # or npm run install
```

### Execute Storybook
```sh
yarn storybook
```

### Compiles and minifies for production
```sh
yarn build
```

### Run your unit tests
```sh
yarn test:unit
```

### Lints and fixes files
```sh
yarn lint
```
